marvis.routing_config

Describes a route config for traffic on the Node.

Classes

RoutingConfig

The RoutingConfig describes a routing config on the Node.

class marvis.routing_config.RoutingConfig(node, dst, gateway)[source]

Bases: object

The RoutingConfig describes a routing config on the Node. At the moment, only ADD operation is supported.

Parameters

  • node (Node) – The node to connect the interface to.

  • dst (str) – The destination IP or subnet, e.g. 192.168.0.0/16.

  • gateway (Interface | str) – The network interface or the gateway IP address to connect to.
